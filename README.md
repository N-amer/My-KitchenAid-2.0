# My-KitchenAid-2.0
Working with different triggers, pieces of logic can be executed.

## Instructions
| Key        | Action                     | 
|------------|----------------------------| 
| ArrowUp    | input value: input + 10    | 
| ArrowDown  | input value: input - 10    | 
| ArrowRight | input value: input + 1     | 
| ArrowLeft  | input value: input - 1     | 
| Q          | input unit: ml            | 
| W          | input unit: cl            | 
| E          | input unit: dl            | 
| R          | input unit: l             | 
| A          | desired unit: ml          | 
| S          | desired unit: cl          | 
| D          | desired unit: dl          | 
| F          | desired unit: l           |
